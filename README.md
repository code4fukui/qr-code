# qr-code

- qr-code tag
- https://code4fukui.github.io/qr-code/

## usage

```html
<script type="module" src="https://code4fukui.github.io/qr-code/qr-code.js"></script>
<qr-code></qr-code>
```

```html
<script type="module" src="https://code4fukui.github.io/qr-code/qr-code.js"></script>
<qr-code>https://code4fukui.github.io/</qr-code>
```
